*** ToastContext

Manages toast notifications (success/error/info messages). Auto-dismisses after 5000ms.

| Export | Line | Type | Purpose |
|--------|------|------|---------|
| ToastType | ToastContext.tsx:3 | Type | 'success' \| 'error' \| 'info' |
| Toast | ToastContext.tsx:5-9 | Interface | { id, message, type } |
| ToastProvider | ToastContext.tsx:19 | Component | Wraps app, manages toasts state |
| useToastContext | ToastContext.tsx:62 | Hook | Returns { toasts, addToast, removeToast } |
| addToast | ToastContext.tsx:42 | Method | (message, type) => adds toast, auto-removes after 5000ms |
| removeToast | ToastContext.tsx:32 | Method | (id) => manually removes toast, clears timeout |

**Usage in App**: App.tsx:39 wraps entire app with ToastProvider.

*** SidebarContext

Manages sidebar collapsed/expanded state. Used by TendersPage to auto-collapse on first tender selection.

| Export | Line | Type | Purpose |
|--------|------|------|---------|
| SidebarContextValue | SidebarContext.tsx:3-8 | Interface | { isCollapsed, toggleSidebar, collapseSidebar, expandSidebar } |
| SidebarProvider | SidebarContext.tsx:16 | Component | Wraps app, manages sidebar state |
| useSidebar | SidebarContext.tsx:41 | Hook | Returns SidebarContextValue, throws if not in provider |
| toggleSidebar | SidebarContext.tsx:19 | Method | () => toggles isCollapsed |
| collapseSidebar | SidebarContext.tsx:23 | Method | () => sets isCollapsed to true |
| expandSidebar | SidebarContext.tsx:27 | Method | () => sets isCollapsed to false |

**Usage in App**: App.tsx:40 wraps app (after ToastProvider) with SidebarProvider.

**Usage in Pages**: TendersPage.tsx:4 imports useSidebar, TendersPage.tsx:9 calls collapseSidebar on first tender click.
